# 商品管理系统

一个使用 HTML、CSS 和 JavaScript 构建的简单商品管理系统，包含以下功能：

- 商品列表展示
- 分页功能
- 搜索功能
- 添加商品
- 编辑商品
- 删除商品

## 功能特点

- 响应式设计，适配移动端和桌面端
- 现代化 UI，使用 Tailwind CSS
- 完整的 CRUD 操作
- 分页和搜索功能

## 部署

该项目已配置 GitHub Actions 自动部署到 GitHub Pages。每次推送代码到 main 分支时，会自动构建并部署到 GitHub Pages。

## 使用方法

1. 在浏览器中打开 `index.html` 文件
2. 使用界面进行商品管理操作

## 技术栈

- HTML5
- Tailwind CSS (CDN)
- JavaScript (ES6+)
